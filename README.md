# To Run Locally
## Clone the project
```bash
  git clone https://github.com/abhay3366/movieApp.git
```

Go to the project directory

```bash
  cd movieApp

```

Install dependencies

```bash
  npm install
```

Start the server


```bash
  npm run dev
```
